LOS TIPOS DE LISTAS


- Lista ordenada numericamente por defecto.

  1. Ejemplo uno
  2. Ejemplo dos
  3. Ejemplo tres

(ejemplo) Lista de alimentos.

  1. Arroz
  2. Frijol
  3. Agua
  4. Lechuga
  5. Bisteck

- Lista ordenada alfabéticamente en minusculas

  1. LETRA A
  2. LETRA B
  3. LETRA C

(ejemplo) Alumnos ordenados de acuerdo a su apellido paterno.

  1. Aranda Cruz Leonardo
  2. Bautista Perez Roberto
  3. Carmilla Gonzales Maria Carmen
  4. Duran Woods Oliver
  5. Eliazar Chavez Michael

- Lista ordenada alfabeticamente en mayusculas

(ejemplo) Nombres ordenados alfabeticamente

  1. Alondra
  2. Brandon
  3. Carlota
  4. Diana
  5. Eunice

- Lista con todos los atributos (numeros romanos)

  1. Elemento
  2. Elemento
  3. Elemento

(Ejemplo) Planetas dentro del sistema solar

  1. Mercurio
  2. Venus
  3. Tierra
  4. Marte
  5. Jupiter
  6. Saturno
  7. Urano
  8. Neptuno
  9. Pluton (no es considerado un planeta pero esta dentro de nuestro sistema solar)

- Lista con todos los atributos (numeros romanos en minusculas)

  1. Chichen itza
  2. Coliseo de roma
  3. Estatua del cristo redentor
  4. Machu pichu
  5. Gran muralla china
  6. Piramides de jiza
  7. Taj Mahal

Lista con elementos desordenados 1 (listas con puntos llenos, lo contrario a las listas 2 de elementos desordenados)

  • Uno
  • Dos
  • Tres
  • Cuatro
  • Cinco
  • Seis
